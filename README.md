# PhotoGallery
Android client side application written in Java 8. Compiled against SDK 26.

Created with i.a. Retrofit2 +  RxJava2  + Java 8

<h2>Features:</h2>

 <ul type="square">
  <li>Preview and edit image EXIF metadata</li>
  <li>Take photos</li>
  <li>Explore gallery</li>
</ul>
